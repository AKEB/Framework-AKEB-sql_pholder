# CurlGet

## Install

composer project akeb/sql_pholder

Composer config

```json
{
    "require": {
        "akeb/sql_pholder": "^1.0.0"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/AKEB/Framework-AKEB-sql_pholder"
        }
    ]
}
```

or

```bash
composer require akeb/sql_pholder
```
